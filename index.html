lunamark - General markup format converter using lpeg.

lunamark is a lua library for converting between markup formats. Currently markdown is the only supported input format, and HTML and LaTeX the only supported output formats, but lunamark's modular structure makes it easy to add new parsers and writers. Parsers are written using a PEG grammar.
